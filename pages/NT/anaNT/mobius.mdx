# Mobius Function

## Definition

The mobius function $\mu$ is defined as follows:

<Card name="Definition">
  <div className=" text-center">$\mu(1)=1$</div>

    If $n > 1$, write $n=p_1^{a_1} \cdots p_k^{a_k}$. Then



    <div className=" text-center m-2">$\mu(n)=(-1)^k$ if  $a_1=\cdots=a_k=1$</div>
    <div className=" text-center m-2">$\mu(n) = 0$ otherwise.</div>

</Card>
