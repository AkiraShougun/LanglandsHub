# Mangoldt Function

## Definition

<Scard name="Definition">
    $\forall n \in \mathbb{Z},n \ge 1$ we define
    ```math
    \Lambda(n) = \ \left\{ \begin{array}{cl}
    \log p & \text{if } n=p^m \text{ for some prime }p \text{ and some }m\ge1,\\
    0 & : \text{otherwise}
    \end{array} \right.
    ``` 
</Scard>
